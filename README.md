Frontend :- React and tailwind css

Backend:- Convex for database

Project has 8 module:-

1)Admin module:- Admin can manage buses, stops, source and destination, seats avialble and prices of each stop.
2)User module:- User can resister and login the system securely as all the credentials are stored as hashes.
3)booking module:- Users can book buses by seeing the aviable buses.
4)Payment Module:- users can do the payment using the provided QRcode.
5)feedback module:- Users can provide the feedback according to there experince.
6)Previews and view buses module:- Users can view the buses information like source and destination, stops, ac/non ac, price,etc. Users can also view the preivew given by other users.
7)About us module:- User can contact the system people and can ask the qeries they have.
8)Avialable in marathi and english both:- Users can use the system has it is avialable in both languages and user can switch the language bu just pressing one button.
The project is deployed on vercel cloud:- https://bus-management-system-eta.vercel.app/

To run the project:-


frontend:- npx convex dev (The backend is also in the frontend folder as there is no other separate backend folder(name as convex) as convex database manages its own backend)

frontend:- npm run dev
